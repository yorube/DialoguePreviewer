# Quick Start

## The loop

1. Open the previewer in any modern browser.
2. Top-right **Text language** → pick your target locale (not `en-US` / `zh-TW`).
3. Left sidebar **Script** → pick a script, click a node.
4. **✏️ Translation Edit Mode** above the dialogue area → hover any line, click ✏️, type your translation, `Ctrl/Cmd + Enter` to save.
5. **💾 Export translation file** at the top when you're done. The file is your save — keep it, re-import next session.

## When something looks off

| What you see | What to try |
| --- | --- |
| English source instead of your translation | Text language is on `en-US`. Switch to your target locale. |
| ✏️ buttons don't appear in Edit Mode | Same reason — switch to your target locale. |
| Export does nothing / says "no baseline" | Source locale, or you've never edited anything. Switch + edit one line. |
| Pill says ⚠️ Unexported edits | Press 💾 Export. To discard, **🔁 Reset this language**. |
| Refreshed and work is gone | If cache wasn't cleared it's still there — keep editing. Otherwise re-import your last exported file. |
| Browser says storage is full | Export, then 🔁 Reset locales you're not actively touching. |

## Keyboard

| Key | Action |
| --- | --- |
| `Space` / `Enter` | Advance one line |
| `1`–`9` | Pick option |
| `R` | Replay current node |
| `←` / `Backspace` | Step back |
| `Ctrl/Cmd + Enter` | Save inline edit |
| `Esc` | Cancel edit / exit Compare Mode |

Shortcuts pause while typing in any input or textarea.
