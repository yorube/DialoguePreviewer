# Quick Start

## The loop

1. Open the previewer in any modern browser.
2. Top-right **Text language**. Pick your target locale (not `en-US` or `zh-TW`).
3. Left sidebar **Script**. Pick a script and click a node.
4. **✏️ Translation edit mode** above the dialogue area. Hover any line, click ✏️, type your translation, save with `Ctrl/Cmd + Enter`.
5. **💾 Export translation file** at the top when you're done. The file is your save. Keep it and re-import next session.

## When something looks off

| What you see | What to try |
| --- | --- |
| English source instead of your translation | Text language is on `en-US`. Switch to your target locale. |
| ✏️ buttons don't appear in Edit mode | Same reason. Switch to your target locale. |
| Export does nothing or says "no baseline" | You're on a source locale, or you've never edited anything. Switch locale and edit one line first. |
| Pill says ⚠️ Unexported edits | Press 💾 Export. To discard, **🔁 Reset this language**. |
| Refreshed and work is gone | If the cache wasn't cleared it's still there, just keep editing. Otherwise re-import your last exported file. |
| Browser says storage is full | Export, then 🔁 Reset locales you're not actively touching. |

## Keyboard

| Key | Action |
| --- | --- |
| `Space` / `Enter` | Advance one line |
| `1`–`9` | Pick option |
| `R` | Replay current node |
| `←` / `Backspace` | Step back |
| `Ctrl/Cmd + Enter` | Save inline edit |
| `Esc` | Cancel edit |

Shortcuts pause while typing in any input or textarea.
